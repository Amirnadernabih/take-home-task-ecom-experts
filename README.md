# take-home-task-ecom-experts

React multi-step bundle builder prototype for the Ecom Experts frontend take-home.

## Phase 1 — Project setup

- Vite + React + TypeScript in the repo root
- Base `src/` structure (`components`, `data`, `hooks`, `styles`, `types`, `utils`)
- Placeholder app: **Bundle Builder** / *Project setup complete*

## Scripts
